
# Modern.IE Virtual Machines

Use the free Windows images from Modern.ie, Vagrant, and Virtualbox to properly test your web applications in IE.

## Available VMs

+ ie6-winxp
+ ie7-vista
+ ie8-win7
+ ie8-winxp
+ ie9-win7
+ ie10-win7
+ ie10-win8
+ ie11-win7
+ ie11-win81

## Usage

```
[~] cd iex-winx
[~/iex-winx] vagrant up
```
